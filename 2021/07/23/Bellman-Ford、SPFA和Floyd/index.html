Bellman Ford、SPFA和Floyd

Bellman-Ford

Bellman-Ford是一种很暴力的最短路算法

擅长解决有边数限制的最短路问题

算法原理：

连续进行松弛，在每次松弛时把每条边都更新一下，若在 n-1 次松弛后还能更新，则说明图中有负环，因此无法得出结果，否则就完成。

需要注意的点

  1. bellman ford算法中，迭代次数k具有实际意义

每轮迭代中的dist数组表示，从起点开始，经过不超过k条边，到达对应点的最短距离

  1. 为了保证上述性质，需要把dist数组进行一个备份，这样防止每次更新的时候出现串联

  2. bellman ford 一般无法在负环的图中使用，但是也有特殊情况

    • 如果负环不在1->n的路径上，或者 题目对经过的边数有限制 还是可以使用Bellman-Ford算法

  3. 由于存在负权边，因此return -1的条件要改成dist[n]>0x3f3f3f3f/2

  4. 为什么无需对重边和自环作单独处理？

    • 重边：由于遍历了所有的边，总会遍历到较短的那一条
    • 自环: 有自环就有自环，反正又不会死循环

  5. 如何利用bellman ford判断负环？

    • 在没有环的图中，n-1次遍历一定可以确定所有点的最短路径
    • 如果图中存在负环，那么在第n次遍历的时候，还会继续绕圈圈，使值更新

  6. 什么时候使用这个算法？

    解决有边数限制的最短路问题。

SPFA

SPFA算法由Bellman-Ford优化而来

在Bellman-Ford算法中，每次松弛所有的点，但是有很多的边遍历了其实没有什么意义，我们只用遍历那些到源点距离变小的点所连接的边即可，只有当一个点的前驱结点更新了，该节点才会得到更新

考虑到这个特性，SPFA进行了优化，每次不松弛所有的点，而只松弛可能更新的点

值得注意的点：

  1. 如何维护距离更新了的点？

    创建一个queue<int> q，如果当前点的距离有更新，就把这个点入队。

  2. ⭐ 为什么返回条件是0x3f3f3f3f而不是bellman-ford算法中的0x3f3f3f3f/2?

    Bellman Ford算法会遍历所有的边，因此不管是不是和源点连通的边它都会得到更新；

    但是SPFA算法不一样，它相当于采用了BFS，因此遍历到的结点都是与源点连通的，因此如果你要求的n和源点不连通，它不会得到更新，还是保持的0x3f3f3f3f。

  3. 🌞 Bellman Ford算法可以存在负权回路，是因为其循环的次数是有限制的因此最终不会发生死循环；

    SPFA算法不可以存在负权回路，由于用了队列来存储，只要发生了更新就会不断的入队，因此假如有负权回路请你不要用SPFA否则会死循环。

  4. SPFA算法仅仅是对Bellman Ford算法进行了优化，假如题目时间允许可以直接用SPFA算法去解Dijkstra算法的题目

    但是如果遇到了不过气的出题人🤬，SPFA可能会被卡掉。

如何使用SPFA来判断负环：

  • 方法 1：统计每个点入队的次数，如果某个点入队n次，则说明存在负环
  • 方法 2：统计当前每个点的最短路中所包含的边数，如果某点的最短路所包含的边数大于等于n，则也说明存在环

852. spfa判断负环 - AcWing题库

代🐎：

  1. 若dist[j] > dist[t] + w[i],则表示从t点走到j点能够让权值变少，因此进行对该点j进行更新，并且对应cnt[j] = cnt[t] + 1,往前走一步

  2. cnt[x] 记录当前x点到源点最短路的边数，初始每个点到源点的距离为0，只要他能再走n步，即cnt[x] >= n，则表示该图中一定存在负环

    • 为什么cnt[x] >= n意味着有负环？

      由于从虚拟源点到x至少经过n条边时，则说明图中至少有n + 1个点，表示一定有点是重复使用，即有负环。

  3. 该题是判断是否存在负环，并非判断是否存在从1开始的负环，因此需要将所有的点都加入队列中，更新周围的点。

Floyd

这个算法是动态规划的思想，时间复杂度O(n^3)

可以求解多源汇最短路

值得注意的点：

  1. 状态转移方程的推导

    g[i, j, k]表示从i走到j的路径上除i和j点外只经过1到k的点的所有路径的最短距离。那么g[i, j, k] = min(g[i, j, k - 1), g[i, k, k - 1] + g[k, j, k - 1]。

    利用滚动数组特性，去除k层后：

    g[i][j] = min(g[i][j], g[i][k] + g[k][j])

  2. 为什么循环k一定要放在最外层？

    因为在计算第k层的f[i, j]的时候必须先将第k - 1层的所有状态计算出来。

  3. 不同于前面的最短路算法使用了dist数组存储最短路径，Floyd算法直接使用在原图g上进行操作，最短路结果也体现在g上。
